1 Instalar node js 22.19.0
2 Instalar y ejecutar xampp para servir la base de datos
3 Crear una base de datos llamada: students_card_db
4 Si tiene clave, colocarla en el archivo src/main/database/database-service.ts al lado del nombre "root"
5 Abrir una terminal en la carpeta del proyecto
6 Ejecutar "npm install" para instalar las dependencias
7 Ejecutar "npm run dev"
